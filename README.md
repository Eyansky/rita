# Rita
- Clone this repo.
```
$ git clone https://github.com/Eyansky/rita.git
```
-Open the project. 
```bash
cd rita
```
- Create a virtual environment
```bash
$ virtualenv env 
```
- start the environment
```bash
$ source env/bin/activate
```
- Install the dependencies
```bash
$ pip install -r requirements.txt
```
- Run the app
```
$ python rita.py
```
